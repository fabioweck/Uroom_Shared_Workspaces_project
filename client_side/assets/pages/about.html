Shared Workspaces Project -
Bow Valley College

WEB PROGRAMMING - WINTER TERM 2023

This page is a fictional business based on shared workspaces for renters and co-workers/users. It was developed and designed by students from Software Development course at Bow Valley College (Calgary/AB - Canada).
The project implements many elements to explore several fields related to Web development, starting from fundamentals such as HTML, CSS and SASS, and going through Javascript, JQuery, Node.js, Express server with customized APIs, and a complete collection of JSON objects in the database generated with SQL to be used as users sample.
All team members collaborated equally to reach the outcome with their unique skills in a well managed and coordinated environment.
